# HappyCapy Customer Service Skill

A Claude Code skill for handling HappyCapy customer support — email replies, product Q&A, and user feedback.

## What This Skill Does

- Detects customer language and replies in kind (Chinese or English)
- Drafts email replies ready to send, no editing needed
- Handles product questions, feedback, bug reports, and feature requests
- Follows anti-spam guidelines to avoid email filters
- Applies consistent tone and signature format

## Files

- `SKILL.md` — The skill definition loaded by Claude Code
- `quick-reference.md` — Fast lookup for common scenarios and tone guidelines

## Usage

Invoke with any of these trigger phrases:

- 客服 / customer service / support
- 回复客户 / reply to customer
- Paste a customer message directly
